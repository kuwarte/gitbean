# gitbean Progress

## Milestones

- [x] `init` — creates .gitbean/ structure (objects/, refs/heads/, refs/tags/, HEAD)
- [x] `hash-object` — hashes + stores blobs, byte-compatible with real git (verified: hashes match)
- [x] `cat-file -p / -t / -s` — reads an object back out, decompresses, parses header, prints content/type/size
- [ ] `add` — staging area / index
- [ ] `write-tree` — binary tree object serialization
- [ ] `commit-tree` / `commit` — commit objects + refs
- [ ] `log` — walk parent pointers
- [ ] `status` — diff working dir vs index vs HEAD
- [ ] `branch` / `checkout` — ref manipulation

## Verified compatibility with real Git

- hash-object: confirmed matching SHA-1 for identical file content (2026-07-15)
- cat-file -p: confirmed real `git cat-file -p <hash>` can read objects written by gitbean (2026-07-15)

## Commands implemented so far

- `gitbean init`
- `gitbean hash-object [-w] <file>`
- `gitbean cat-file -p <hash>` — print content
- `gitbean cat-file -t <hash>` — print type
- `gitbean cat-file -s <hash>` — print size
